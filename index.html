Projects

Daily Habit Tracker

Recently, I have read about the power of habits and how tracking them will improve the likelihood of forming new habits. Using HTML forms, my app allows you to input a new habit you want to start tracking, and every day the app will ask you Yes or No if you did the habit, for all the habits you have inputed. Hockey Lesson Scheduler

On my hockey coaching website, I used JavaScript to create a matchmaking tool that will match a player with one of our coaches based on availability and skill level. From there, I used local hockey rink schedule APIs to give the user available ice times and locations to book a lesson with their Matching coach!
